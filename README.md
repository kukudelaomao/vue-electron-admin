# vue-electron-admin

> 基于vue和electron，快速构建中后台页面，融合了vue-element-admin

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


# lint all JS/Vue component files in `src/`
npm run lint

```
